# Prédiction du prix des appareils d'occasion

Ce projet utilise le **Machine Learning** (régression linéaire) pour estimer le prix de vente de téléphones et tablettes de seconde main.

## Objectif
Analyser les caractéristiques techniques (Batterie, Caméra, RAM, Année) pour prédire une valeur de marché juste.

## Stack Technique
* **Langage :** Python
* **Librairies :** Pandas, Scikit-Learn, Matplotlib & Seaborn.

## Étapes Clés
1. **Nettoyage des données :** Gestion des valeurs manquantes et doublons.
2. **Preprocessing :** Encodage des variables (Marques, OS).
3. **Modélisation :** Entraînement d'un modèle de régression pour la prédiction numérique.

## Utilisation
Ouvrez le fichier `.ipynb` dans Jupyter ou Google Colab pour visualiser l'analyse et les résultats.
